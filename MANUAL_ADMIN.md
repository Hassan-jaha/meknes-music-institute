# 📖 Manuel d'Utilisation - Espace Administration

Bienvenue dans l'interface de gestion de l'Institut de Musique de Meknès. Ce guide vous explique comment gérer le contenu de votre site.

## 🔑 1. Accès à l'Administration
1. Rendez-vous sur la page `Admin` (en haut à droite du site).
2. Connectez-vous avec vos identifiants.
3. Vous arrivez sur le **Tableau de Bord** qui résume l'activité de votre site.

## 📰 2. Gestion des Actualités
Pour ajouter une actualité :
1. Cliquez sur **Actualités** dans le menu ou le bouton sur le tableau de bord.
2. Cliquez sur **Ajouter**.
3. Remplissez le titre, la date et le contenu.
4. Sélectionnez une image (elle sera automatiquement redimensionnée).
5. Cliquez sur **Enregistrer**.

## 📢 3. Gestion des Annonces
Les annonces apparaissent dans le bandeau doré en haut du site si elles sont **Épinglées**.
1. Allez dans la section **Annonces**.
2. Vous pouvez créer une annonce et cocher l'option **Épinglé** pour qu'elle soit visible partout immédiatement.

## 🖼️ 4. Gestion de la Galerie
1. Allez dans **Galerie**.
2. Ajoutez une photo en fournissant une petite description.
3. Les photos s'affichent automatiquement sur la page publique Galerie.

## 📩 5. Consultation des Messages
Chaque fois qu'un visiteur remplit le formulaire de contact :
1. Une notification (chiffre) s'affiche sur la carte **Messages** du tableau de bord.
2. Cliquez sur **Consulter les Messages**.
3. Vous pouvez lire le contenu et l'email de l'expéditeur pour lui répondre.
4. Une fois traité, vous pouvez supprimer le message.

## 🌐 6. Multilingue
L'interface d'administration s'adapte automatiquement à la langue sélectionnée. Si vous changez la langue en haut du site, les menus de l'admin changeront également pour plus de confort.

---
*Note : Pour des raisons de sécurité, n'oubliez pas de vous déconnecter après chaque session.*
